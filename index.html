Bird strike reports 2017-2021

Introduction

The dataset contains birdstrikes on airplanes from 1990-2022. There are 100 columns describing things as size, time of day, airport etc. Also, there are 267.000 rows with reviews. The dataset therefore contains nominal values since we see the categories in the columns. To get the right answers we also used aggregate columns such as AVG, COUNT etc.

If we take a look on some of the different types of data, we can see that it mainly contains primitive data, rather than non-primitive data.


Primitive data – examples

  • Numbers
  • Strings in the form of names, such as airport names, time of day etc.
  • An entire column has complex data, as we get a lot of mixed strings
  • In the dataset there is used NULL

Non-primitive data – example

  • In the charts there is used arrays

The quality of the dataset is not that good. We get a lot of information about the birdstrikes such as were it happened, the time, the airport but there is also a lot of missing information. If we take a look at the column ‘aircraft’ there is a lot of places where it is saying that “?” which makes the accuracy of the data worse.

Overall the data doesn’t seem very accurate, since there is a lot of NULL in the dataset. There is a lot of missing spots but seems more fulfilled as the years go by.

The data is collected thru FAA’s 5200-7 form. Pilots must fill out a detailed form after encountering a wildlife strike. The data is then recorded in FAA’s Wildlife strike database. Since the data is collected thru forms, there might be accuracy issues.
